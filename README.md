# GPSTRACKER - Приложение для гео-напоминаний

Приложение на Java для Android, которое позволяет устанавливать напоминания, привязанные к конкретным точкам на карте. Уведомление срабатывает, когда вы входите в радиус 100 метров от созданной метки.

## Основные функции
- **Отображение позиции:** Синяя метка показывает ваше текущее местоположение на карте.
- **Создание точки:** Введите текст в поле сверху и нажмите на любое место на карте.
- **Радиус срабатывания:** Автоматическая зона 100 метров вокруг каждой метки.
- **Удаление:** Удерживайте палец на метке (Long Press), чтобы удалить её.
- **Push-уведомления:** Приложение присылает уведомление со звуком и вибрацией при входе в зону.

## Технологии
- **Язык:** Java
- **Карты:** osmdroid (OpenStreetMap)
- **Геолокация:** Google FusedLocationProvider (High Accuracy)
- **Минимальная версия:** Android 7.0 (API 24)

## Установка
1. Клонируйте репозиторий.
2. Откройте проект в Android Studio.
3. Убедитесь, что в файле `build.gradle.kts` добавлены зависимости:
   - `org.osmdroid:osmdroid-android:6.1.18`
   - `com.google.android.gms:play-services-location:21.1.0`
4. Соберите и запустите проект на устройстве с поддержкой GPS.

## Разрешения
Для корректной работы приложению требуются:
- Доступ к точной геолокации (`ACCESS_FINE_LOCATION`).
- Доступ к интернету (для загрузки карт).
- Разрешение на отправку уведомлений (для Android 13+).
